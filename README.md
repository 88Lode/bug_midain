# Bug Visualizer Portfolio

A premium, high-fidelity static showcase for the **Bug Visualizer** asset. This project was developed as part of a technical portfolio for Midain Studios.

## Features
- **Modern UI**: Clean, dark-themed aesthetic with glassmorphism effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Micro-animations**: Smooth reveal transitions and interactive hover states.

## Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Vanilla, custom animations)
- JavaScript (Intersection Observer for reveal effects)
